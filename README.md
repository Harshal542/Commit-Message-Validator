# Git Commit Validator

A Git `commit-msg` hook built in Python that enforces the Conventional Commits standard. It includes a native UI popup to alert developers if their commit message is formatted incorrectly.

## How to Install
1. Clone this repository.
2. Run `python setup.py` to automatically install the hook.

## Allowed Commit Types
* `feat:` (New features)
* `fix:` (Bug fixes)
* `docs:` (Documentation)
* `style:` (Formatting)
* `refactor:` (Code improvements)
* `test:` (Unit tests)
* `chore:` (Maintenance)
